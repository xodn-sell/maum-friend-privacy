# 개인정보처리방침

**마음친구** (이하 "본 앱")는 사용자의 개인정보를 소중히 여기며, 다음과 같이 처리합니다.

## 1. 수집하는 개인정보 항목

본 앱은 다음의 개인정보를 수집합니다:
- **필수 정보**: 닉네임, 나이
- **선택 정보**: 관심사, 대화 내용, 기분 기록

## 2. 개인정보의 수집 및 이용 목적

수집한 개인정보는 다음의 목적으로만 이용됩니다:
- AI 기반 심리 상담 서비스 제공
- 맞춤형 대화 경험 제공
- 기분 추적 및 분석

## 3. 개인정보의 보관 및 저장

- **로컬 저장**: 모든 대화 내역과 기분 기록은 사용자의 기기에만 저장됩니다.
- **외부 전송**: 대화 내용은 AI 응답 생성을 위해 Groq API 서버로 전송되며, 서버에 영구 저장되지 않습니다.
- **보관 기간**: 사용자가 앱을 삭제하거나 데이터를 직접 삭제할 때까지 보관됩니다.

## 4. 개인정보의 제3자 제공

본 앱은 다음의 경우를 제외하고 사용자의 개인정보를 외부에 제공하지 않습니다:
- **AI 서비스 제공**: Groq API (대화 생성 목적)
- **광고 서비스**: Google AdMob (맞춤형 광고 제공)

## 5. 개인정보의 파기

- 사용자가 앱을 삭제하면 기기에 저장된 모든 데이터가 자동으로 삭제됩니다.
- 사용자는 앱 내 설정에서 언제든지 데이터를 삭제할 수 있습니다.

## 6. 사용자의 권리

사용자는 다음의 권리를 가집니다:
- 개인정보 열람 요구
- 개인정보 수정 요구
- 개인정보 삭제 요구
- 개인정보 처리 정지 요구
